NAME

       pfuShowTextureMemory,  pfuClearTextureMemory  -  Utilities
       for texture memory on Infinte Reality.

FUNCTION SPECIFICATION

       #include <Performer/pfutil.h>

       void   pfuShowTextureMemory(PipeWindow *pw,
                pfList *texList, int format);

       void   pfuClearTextureMemory(void);

DESCRIPTION

       Get  a  snap  shot  of Infinte Reality texture memory with
       pfuShowTextureMemory  and  clear   texture   memory   with
       pfuClearTextureMemory.

       pw  is  the  pfPipeWindow  where the texture banks will be
       displayed.

       texList is a list of all textures thought to be loaded  in
       texture memory, such as returned by pfuMakeTexList.



       format may be any of PFTEX_RGB_5 PFTEX_RGB_4 PFTEX_RGB5_A1
       PFTEX_RGBA_4 PFTEX_IA_8 PFTEX_I_12A_4 PFTEX_I_8 PFTEX_I_16
       PFTEX_IA_12 PFTEX_RGBA_8 PFTEX_RGB_12 PFTEX_RGBA_12
