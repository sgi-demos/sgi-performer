DESCRIPTION

       pfiCollide functions.

       The pfiCollide has a complete C++ API following the con-
       ventions of general OpenGL Performer C++ API; C++ methods
       are declared in /usr/include/Performer/pfui/pfiCollide.h.

       pfiInit should be called once before any pfiCollide rou-
       tines and before the forked creation of any additional
       processes that will be calling pfiCollide routines.

NOTES

       The libpfui source code, object code and documentation are
       provided as unsupported software.  Routines are subject to
       change in future releases.
