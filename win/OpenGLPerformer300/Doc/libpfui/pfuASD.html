NAME

       pfuFindASDs  -  Collection of all pfASD nodes in the scene
       graph

FUNCTION SPECIFICATION

       #include <Performer/pfutil.h>

       void   pfuFindASDs(pfNode *node, pfList *ASDs);

DESCRIPTION

       pfuFindASDs finds all  pfASD  nodes  in  the  scene  graph
       rooted by node and store them in pfList ASDs.  ASDs should
       be initialized before passing to this function.

NOTES

       The libpfutil source code, object code  and  documentation
       are  provided  as unsupported software.  Routines are sub-
       ject to change in future releases.
