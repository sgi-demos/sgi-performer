NAME

       pfGetId - Get unique id of libpf object.

FUNCTION SPECIFICATION

       #include <Performer/pf.h>

       int   pfGetId(void *unknown);

PARAMETERS

       unknown  is a pointer to a libpf object

DESCRIPTION

       All  OpenGL Performer objects defined in the libpf library
       have a unique integer  identifier.   pfGetId  returns  the
       identifier of unknown or -1 if unknown is not a libpf data
       type.
