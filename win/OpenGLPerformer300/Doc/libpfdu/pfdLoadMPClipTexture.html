NAME

       pfdLoadMPClipTexture  - Read Configuration File to Config-
       ure a MultiProcess Clip Texture

FUNCTION SPECIFICATION

       #include <Performer/pfdu.h>

       pfMPClipTexture
                           *   pfdLoadMPClipTexture(const char *fileName)

DESCRIPTION

       pfdLoadMPClipTexture accepts a string containing the  path
       to  a  configuration  file  describing a pfClipTexture. It
       configures and creates a multiprocess clip texture.

       A Multiprocess Clip Texture is part of libpf, and unlike a
       Clip  Texture,  is  not restricted to use in the draw pro-
       cess. See the pfMPClipTexture man page for  more  informa-
       tion.

       The configuration files for creating a pfClipTexture and a
       pfMPClipTexture are identical. See the  pfClipTexture  man
       page  for  information  on  the clip texture configuration
       file format.

NOTES

       ClipTexture functionality is not supported under Linux.

BUGS

       The parser that reads configuration files is very limited.
       In  particular,  it only allows comments where it would be
       expecting a token, and expects the tokens and data  fields
       in a fixed order.
