NAME

       pfdASDClipring  -  Construct pfASD clipring size and value
       for virtual cliptexture

FUNCTION SPECIFICATION

       #include <Performer/pfdu.h>

       void   pfdASDClipring(pfASD *asd, int numrings);

DESCRIPTION

       pfdASDClipring computes numrings level of pfLODRange based
       on the original asd LODRange and enables clipring in pfASD
       node.

       See libpfdb/pfevt.c  for  example.   Set  _PFASD_CLIPRINGS
       without  any  particular  value will turn on the automatic
       generation of number of cliprings.   Set  _PFASD_CLIPRINGS
       to  an int, then pfdASDClipring will build that many addi-
       tional LODRanges inside the smallest  switchin  range  and
       use them as clipring boundries.

       The libpfdu source code, object code and documentation are
       provided as unsupported software.  Routines are subject to
       change in future releases.
